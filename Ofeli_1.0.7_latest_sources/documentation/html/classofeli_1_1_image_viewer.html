Detailed Description

The class ImageViewer is a graphical user interface using the Qt4 framework (QtCore and QtGui) to show an image with the active contour, to configure the parameters of the active contour and optionally the preprocessing step and to evaluate two segmentation results thanks to the evaluation window. It based on the Qt example of the documentation http://doc.qt.nokia.com/4.7/widgets-imageviewer.html.
